# Crisis Responder

A real-time crisis orchestration platform for hospitality (hotels, resorts, restaurants).

## System Architecture
1. **Backend**: Node.js + Express + Firebase
2. **Guest Web**: Next.js (QR Portal)
3. **Staff App**: React Native (Mobile App)

## Setup Instructions

### Backend Setup
1. Navigate to `/backend`.
2. Run `npm install` (once dependencies are finalized).
3. Copy `.env.example` to `.env` and fill in your Firebase and Gemini API credentials.
4. Run `npm run dev` to start the backend server on port 3000.
